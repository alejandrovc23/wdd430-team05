## Project Decisions

The repository defines Handcrafted Haven as an e-commerce platform intended to connect artisans and crafters with customers seeking unique handmade products. The current semester direction includes seller profiles and stories, product discovery, catalog filtering and sorting, ratings, written reviews, accessibility, responsive design, and a future full-stack implementation. Week 02 is limited to setup, planning, design identity, reusable starter components, and the application foundation. These repository-defined decisions still require confirmation against the actual meeting record.

## Design Decisions

The initial design direction recorded in the repository includes:

- A palette of Forest (`#254441`), Forest Dark (`#17312F`), Terracotta (`#A34A2A`), Cream (`#FFF8EE`), White (`#FFFFFF`), Charcoal (`#2D2A26`), and Sage (`#8BAA91`).
- Georgia, Cambria, and Times-family serif typefaces for headings.
- Arial, Helvetica, and sans-serif typefaces for body and interface text.
- A mobile-first layout with responsive card grids.
- A WCAG 2.1 Level AA accessibility goal, semantic landmarks, visible focus states, alternative text, and reduced-motion support.
- Reusable header, hero, category-card, product-card, artisan-section, and footer components.
